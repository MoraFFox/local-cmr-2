# Quick Start: PDF Export

## Installation Complete ✅

The professional PDF export system has been installed and configured.

## What Changed?

### Before:
- Browser print → Screenshot-like PDFs
- Poor formatting
- Single version only

### After:
- Professional PDF generation
- Two versions: **Internal** (with costs) & **Client** (without costs)
- Well-formatted tables and sections
- Automatic page breaks and numbering

## How to Use

### Export Full Company Report:
1. Go to **History** page
2. Click on any company to view details
3. Click **"Export Full Report"** button (top right)
4. Choose:
   - **Internal Report** → Includes all costs (for your company)
   - **Client Report** → Hides costs (to send to clients)
5. PDF downloads automatically

### Export Single Branch Report:
1. In company details, find the branch you want
2. Click to expand the branch card
3. Click **"Print Branch"** button
4. Choose Internal or Client version
5. Branch-specific PDF downloads

## What's Included in PDFs?

### Full Company Report:
- Company profile (name, tax ID, email, location)
- All contacts with phone numbers
- Machine ownership details
- **If no branches:** Main office maintenance history
- **If has branches:** Each branch with:
  - Branch info and contacts
  - Assigned staff with ratings
  - Complete maintenance history
  - Parts replaced summary

### Branch Report:
- Branch information
- Branch contacts
- Assigned staff (with ratings and notes)
- Detailed maintenance history
- Machine status
- All services and parts

## Internal vs Client Reports

| Feature | Internal Report | Client Report |
|---------|----------------|---------------|
| Maintenance details | ✅ | ✅ |
| Staff names | ✅ | ✅ |
| Services performed | ✅ | ✅ |
| Parts replaced | ✅ | ✅ |
| **Lease costs** | ✅ | ❌ |
| **Part costs** | ✅ | ❌ |
| **Service costs** | ✅ | ❌ |
| Footer | "CONFIDENTIAL" | "Service Report" |

## File Names

PDFs are automatically named:
- `CompanyName_Internal_Report_2024-01-15.pdf`
- `CompanyName_Client_Report_2024-01-15.pdf`
- `CompanyName_BranchName_Internal_Report_2024-01-15.pdf`

## Tips

1. **Use Internal Reports** for your company records and financial tracking
2. **Use Client Reports** when sending maintenance summaries to clients
3. **Branch Reports** are perfect for location-specific documentation
4. PDFs are ready to email or print immediately
5. All data is properly formatted in professional tables
